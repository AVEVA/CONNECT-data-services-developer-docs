    Community Search

    Defines the public API endpoints that are used to search communities. A community provides a way to share information, such as data streams, between customers.

    Community Summary Information

    Gets community summary information

    Request

    GET /api/v1-preview/tenants/{tenantId}/communities/{communityId}/summary

    Parameters

    string tenantId
    Calling tenant identifier

    string communityId
    Community identifier

    Response

    Status Code Body Type Description
    200 CommunitySummaryInformation Returns community summary information of type CommunitySummaryInformation.
    400 ErrorResponse Bad Request. The server could not understand the request.
    401 None Unauthorized. The client has not been authenticated.
    403 ErrorResponse Forbidden. The client does not have the required permissions to make the request.
    404 ErrorResponse Not Found. The requested tenant or community was not found.
    408 ErrorResponse Request Timeout. The request has timed out.
    500 ErrorResponse Internal Server Error. The server has encountered a situation it doesn't know how to handle.

    Example response body

    200 Response

    {
  "TotalStreams": 0,
  "StreamsContributed": 0
}

    Authorization

    Allowed for these roles:

    • Community Administrator
    • Community Member
    • Community Moderator
    • Tenant Member

    Search Streams

    Searches for streams within a community by query

    Request

    GET /api/v1-preview/tenants/{tenantId}/search/communities/{communityId}/streams
?query={query}&count={count}&searchTenantId={searchTenantId}

    Parameters

    string tenantId
    Calling tenant identifier

    string communityId
    Community identifier

    [optional] string query
    Query to execute. The query uses the same format as SDS. See Search in SDS.

    [optional] integer count
    Maximum total results

    [optional] string searchTenantId
    The tenant identifier of the streams to be searched. This parameter enables the API to search streams based on tenantID. By default this parameter holds an empty GUID.

    Response

    Status Code Body Type Description
    200 StreamSearchResult Returns the stream information that matches the search criteria. This is a set of objects of type StreamSearchResult.
    400 ErrorResponse Bad Request. The server could not understand the request due to invalid syntax.
    401 None Unauthorized. The client has not been authenticated.
    403 ErrorResponse Forbidden. The client does not have the required permissions to make the request.
    404 ErrorResponse Not Found. The requested community was not found.
    408 ErrorResponse Request Timeout. The request has timed out.
    500 ErrorResponse Internal Server Error. The server has encountered a situation it does not know how to handle.

    Example response body

    200 Response

    {
  "Id": "string",
  "Name": "string",
  "TypeId": "string",
  "Description": "string",
  "Self": "string"
}

    Authorization

    Allowed for these roles:

    • Community Administrator
    • Community Member
    • Community Moderator
    • Tenant Member
